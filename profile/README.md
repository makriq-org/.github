## makriq-org

Публичные репозитории вокруг VPN-сервиса.

- [`FlClashM`](https://github.com/makriq-org/FlClashM) — Android-клиент для `mihomo`
- [`sys-cfg`](https://github.com/makriq-org/sys-cfg) — публичные rulesets, списки и redirector
- [`short-key-list`](https://github.com/makriq-org/short-key-list) — проверка и отбор публичных VLESS-ключей
